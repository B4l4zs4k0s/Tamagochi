TAMAGOCHI 

This basic HTML game is representing a "Club of Foxes". (Foxes are stored in a simple list)

You can create a new fox on the "/login" endpoint, or by clicking on the "Change Fox" link in the found in the information page.

The game has some basic functions like:

Nutrition store: Here you can choose what your fox is feeding on.
Trick center: Where you can teach your fox new tricks.

These changes will show up on the fox's information page.

This game was made to practice HTML, Thymeleaf, endpoints, the way annotations work and how to write basic repository and services.
